G: Gathering Diamonds

﻿You are given an NxM grid in which some spaces contain a diamond and the other spaces are empty. You start in the bottom far-left space, and must advance to the goal in the top far-right space, collecting all of the diamonds on the way. You can only move one space left, right, or up in a single step. By stepping on a diamond, you automatically acquire it. Your objective is to reach the goal with all of the diamonds in the minimum number of steps.

Input:

  1. The first line contains T (1≦T≦50), the number of test cases.
  2. T test cases follow.
  3. Each test case begins with a line giving N and M (1≦N,M≦300), the height and width of the board.
  4. N lines follow, each containing M ‘.’ or ‘o’ characters. These describe the board from top to bottom.
    1. ‘.’ represents an empty space
    2. ‘o’ represents a diamond

Output:

For each test case, output the minimum number of steps required to collect all diamonds and reach the goal, followed by a new line.

Sample input:

3
7 8
......o.
o.o..o..
.o..o...
..o.....
.o...o..
........
...o....
3 3
..o
.o.
o..
2 2
..
..

Sample output:

29
4
2
