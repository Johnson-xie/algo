E: Elections

﻿John has decided to run for president of his high school computer science club. The voters in the class have compiled rankings of the candidates they like. They will all vote for the earliest candidate on their list that is running. John has managed to acquire all of these lists. By using his charm and charisma, John can convince K candidates to stop running for election. What is the maximum number of votes that John can receive?

Each candidate has a number assigned to them, and John is candidate number 1. We can represent each voter's preferences as an ordered list. If a voter's preferences are "2 1 3", that means John is their second choice -- after candidate #2, and before candidate #3.

Input:

  1. The first line gives T (1≦T≦100), the number of test cases
  2. T test cases follow
  3. Each test case begins with three numbers, M N K (M * N ≦ 256).
    1. M is the number of candidates
    2. N is the number of voters
    3. K is the maximum number of individuals John can convince to not run
  4. This is followed by an N lines containing M space-separated integers, where the ith line is the ith voter’s order of preferences.

Output:

For each test case, output one line containing the maximum number of votes John can receive, followed by a new line.
