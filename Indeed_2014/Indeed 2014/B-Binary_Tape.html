B: Binary Tape

﻿You have a binary tape, with 0s and 1s written on it. You can cut the tape in between any digits to result in a larger number of shorter binary tapes. The value of a piece of tape is defined as its interpretation as an unsigned base two integer.

We also define a valid segment to be segment of tape whose value is at least N. Given a piece of tape and a positive integer N, what is the largest number of valid segments you can cut it into?

Input:

  1. The first line gives T (1≦T≦100), the number of test cases.
  2. T test cases follow.
  3. Each test case consists of two lines. The first line gives N, the minimum value of a valid piece of tape. The second line gives S, a binary string of length between 1 and 100 digits, representing the tape.

Output:

For each test case, output the maximum number of valid segments the tape can be cut in to, followed by a new line.

Sample input:

5
32
10101100110011000101101010110110
1
1010101010111010101010101
99
11111111111101111111111111110111111111110111111111111
18
111110111110110100000000000000011111111101011111111000101
3
10

Sample output:

4
14
7
7
0

Explanation:

10101100110011000101101010110110

Cut optimally so that its the most pieces. If cut as the above: 1. 101011 2. 00110011 3. 000101101 4. 010110110 Four valid pieces of tape can be created.
