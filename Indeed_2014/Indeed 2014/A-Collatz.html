A: Collatz

﻿Consider the following function:

Given a positive integer X:

  • If X is even, X, divide X by 2.
  • If X is odd, triple X and add 1.

The stopping time of positive integer N is defined by the number of repetitions of this function to eventually reach the number 1. For example, the stopping time of 1 is 0 and the stopping time of 5 is 5 (5 -> 16 -> 8 -> 4 -> 2 -> 1).

The unproven Collatz Conjecture speculates that for every positive integer N, the process will eventually reach the number 1 (that is, every positive integer has a well-defined stopping time).

Given an integer N, output the stopping time of N.

Input:

  1. The first line gives T (1≦T≦100), the number of test cases.
  2. T lines follow, each giving input N (2≦N≦10,000)

Output:

For each test case, output the stopping time of N, followed by a new line.

Sample input:

4
3
47
2
10000

Sample output:

7
104
1
29
