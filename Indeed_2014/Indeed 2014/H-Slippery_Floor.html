H: Slippery Floor

﻿Tiles are arranged in a NxM grid. Because the surface of these tiles is so slippery, if you walk in a direction you will continue moving in that direction until you collide with an obstacle. Your task is to determine the number of combinations of start and end tiles such that you cannot reach (even in passing) the end tile from the start tile.

Input:

  1. The first line gives T (1≦T≦100), the number of test cases.
  2. T test cases follow.
  3. Each test case begins with a line containing N and M, the height and width of the room.
  4. 1≦N,M≦50
  5. The next N lines give the layout of the room, with each line containing M characters indicating what occupies that position in the room
    1. ‘.’ represents an empty tile
    2. ‘#’ represents an obstacle
  6. The room is guaranteed to be bordered by obstacles.

Output:

For each test case, output the number of combinations of start and end tiles such that you cannot reach the end tile from the start tile, followed by a new line.

Sample input:

3
7 8
########
#...#..#
#...####
#......#
###....#
###....#
########
5 5
#####
#.#.#
#####
#.#.#
#####
4 8
########
#......#
#...#..#
########

Sample output:

123
12
0
