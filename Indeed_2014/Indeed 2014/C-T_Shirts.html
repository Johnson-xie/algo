C: T-Shirts

﻿There are shirts of three different sizes, small (S), medium (M), and large (L). People also have size preferences corresponding to each shirt size.

  • People who prefer S can wear all 3 sizes.
  • People who prefer M can wear sizes M and L.
  • People who prefer L can only wear size L.

Given a set of preferences and a set of shirts, determine whether everyone can be given a shirt that they can wear.

Input:

  1. The first line gives T (1≦T≦100), the number of test cases.
  2. T test cases follow.
  3. The first line of each test case gives the collection of shirt sizes available.
  4. The second line of each test case gives the collection of personal size preferences.
  5. The number of shirts and the number of people are always equal (call this number N), and 1≦N≦100.

Output:

For each test case, output YES if everyone can be given a shirt, or NO otherwise. Output a new line after each test case.

Sample input:

3
MMSLLLLLLSMS
LSMMSLMSLLMS
LLLLLLLLLLLLLL
SSSSSSSSSSSSSS
LLLSSLLLSSLLLLSS
MSLLMLMSLMLMLMSM

Sample output:

YES
YES
NO
