I: Tower of coins

﻿You are given a number of turns t, in which you can attempt to stack coins. You get to keep the number of coins that are stacked at the end of your turns. Let's assume that in one turn, you can stack 1 coin. The chance of successfully stacking the kth coin is 1/k.

  • If you succeed, the number of coins increases by 1
  • If you fail, the number of coins returns to 0

At each turn, you can either attempt to add a coin to the tower, or do nothing.

You want to maximize the number of coins you keep. Determine the expected value of this quantity.

Input:

  1. The first line gives T (1≦T≦50), the number of test cases.
  2. T test cases follow.
  3. Each test case consists of one line giving t (1≦t≦1011), the number of turns.

Output:

For each test case, output the expected value of the optimal strategy, followed by a new line.

Sample input:

5
7
3
94
1
1234

Sample output:

1.875
1.5
3.5808567386446
1
5.0803515513296
